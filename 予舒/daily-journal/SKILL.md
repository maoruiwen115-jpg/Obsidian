# daily-journal 日报生成器

## 核心原则

- **分段写盘**：每段生成后立即写盘，不累积内容
- **搜索即提炼**：搜索完成立即提炼关键点，原始结果丢弃
- **人物不重复**：写人物传记前必须查索引，从不重复推荐

## 日报结构

```
## 📅 每日日报 | YYYY年MM月DD日

## 🌟 今日核心（必读）
- AI新闻摘要（2-3句）
- 今日人物 + AI热点关联（1段）

## 📋 昨日回顾
- 简短回顾昨天日报核心 + 今日关联

## 💡 极简主义学习
（来源：The Minimalists，每日提取1个核心观点）

## 🤖 AI新闻 × 3
（搜索，来源优先36氪/机器之心/量子位）

## 📖 人物传记推荐
### 一、时间轴：关键人生节点
### 二、核心哲学（3条）
### 三、金句（3-4条）
### 四、普通人能学到什么（3-5条）
### 五、写在最后（情感升华）
### 六、相关书籍推荐（2-3本）

## 🙏 感恩时刻
```

## 执行流程（5次写盘）

### Step 1：准备
```bash
# 读陪伴天数
cat ~/.openclaw/workspace-dev/.counter.md 2>/dev/null || echo "1"

# 查天气
curl -s "wttr.in/Changsha?format=3"
```

### Step 2：搜索AI新闻
```
web_search({ query: "AI 人工智能 最新新闻 YYYY年MM月DD日", count: 3, maxChars: 500 })
```

### Step 3：搜索极简主义
```
web_search({ query: "site:theminimalists.com minimalism 2026", count: 1 })
```

### Step 4：分5次写盘

| 次序 | 内容 | 约字数 |
|------|------|--------|
| 1 | 标题+今日核心+昨日回顾 | 300字 |
| 2 | 极简主义学习 | 400字 |
| 3 | AI新闻×3 | 800字 |
| 4 | 人物传记 | 1500字 |
| 5 | 结尾+感恩 | 300字 |

### Step 5：更新传记索引
在索引文件中添加一行：`| YYYY-MM-DD | 人物名 | 核心主题 |`
